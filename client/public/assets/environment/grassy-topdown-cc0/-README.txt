Thank you for downloading this tileset!

Enclosed in this folder you will find:

- a PNG of the tileset
- the original Aseprite file for this tileset
- a guide for setting the tileset up as an autotile/terrain set

The guide is made with the Godot Engine's autotile/terrain system in mind.
It may work for other game engines, though since I (the creator) have no experience with any other engines I cannot say for sure

This tileset is released under the CC0 license, meaning no permission is needed to use it for your projects, commercial or otherwise.

Read more at https://creativecommons.org/publicdomain/zero/1.0/

While no credit or donation is required, it would still be greatly appreciated, especially if this tileset is used for a commercial project.
