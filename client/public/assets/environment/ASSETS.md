# Подбор ассетов окружения Clayfolk

Эта папка - рабочая заготовка внешнего мира для поверхности: теплая трава, песочная земля, вода, камни, кусты, деревья, костер и лагерные мелочи в духе текущего референса Clayfolk. Человечки здесь не берутся: персонажей делаем своими.

Ассеты распакованы в обычные папки и могут подключаться клиентским рендером напрямую. Сырые zip-архивы из `_downloads/` удалены из репозитория.

## Основной набор

- `summer-plains/` - главный кандидат под базовую карту: теплые 32x32 тайлы травы, земли, воды, деревья, кусты, камни, цветы, костер, палатка, мостик и мелкий декор.
- `summer-plains-water-animation-demo/` - демо анимации воды из того же набора.

Источник: https://schwarnhild.itch.io/summer-plains-tileset-and-asset-pack-32x32-pixels

Лицензия по странице автора: можно использовать в коммерческих и некоммерческих проектах, можно изменять, нельзя перепродавать или распространять сами ассеты отдельным паком; кредит желателен.

Текущий рендер берёт из `summer-plains/summer_plains_v1.0_standard/assets.png` три больших дерева для края карты. Их нужно резать как цельные объекты по прозрачной границе, а не по условной сетке:

- `treeTall`: `crop(assetsUrl, 0, 0, 160, 240)`;
- `treeRound`: `crop(assetsUrl, 176, 34, 164, 210)`;
- `treeWide`: `crop(assetsUrl, 360, 26, 152, 218)`.

Подробный метод нарезки и расстановки записан в `docs/Метод переходов поверхности.md`.

## Дополнительные кандидаты

- `epic-rpg-world-grassland-demo/` - хороший набор лагерных и природных деталей: камни, деревья, земля, трава, пропсы, sample scene. По настроению близко к референсу, но это free demo, не полный платный набор.

Источник: https://rafaelmatos.itch.io/epic-rpg-world-free-demograss-land-20

Лицензия по странице автора: можно использовать в личных и коммерческих проектах, можно изменять; нельзя переупаковывать, перепродавать или распространять сами ассеты; нельзя использовать для NFT и обучения AI.

- `topdown-fantasy-forest/` - запасной лесной набор 16x16: деревья, кусты, камни, вода, дорожки, декор с тенями и без теней.

Источник: https://aamatniekss.itch.io/topdown-fantasy-forest

Лицензия по странице автора: можно использовать бесплатно, в том числе коммерчески, можно изменять; кредит не обязателен, но приветствуется; нельзя перепродавать/распространять как отдельный ассет-пак, нельзя использовать для NFT/AI.

- `grassy-topdown-cc0/` - простой CC0-набор: трава, песок/светлая земля, вода. Полезен как безопасная базовая подложка или источник отдельных тайлов.

Источник: https://ringosnoop.itch.io/grassy-top-down-tileset

Лицензия по странице автора: CC0, кредит не требуется.

- `forest-tent-tileset/` - маленький набор лагерных PNG: тайлсет/спрайты и анимированный костер. Подходит как быстрый источник костра, палатки, бревен и мелкого лагерного декора.

Источник: https://thomaswastaken.itch.io/tileset

Лицензия по странице автора: royalty-free, можно использовать в любом движке; автор просит указать кредит, если это возможно.
